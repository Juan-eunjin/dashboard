# dashboard
Backend 코드

세팅 : Spring Boot 3 + MyBatis + PostgreSQL



[진행 예정]
- **API 연동 및 데이터 수집 모듈 개발:** JIRA API를 호출하여 데이터를 가져오는 기본 통신 모듈을 구축합니다.
- **실시간 이벤트 수신(Webhook) 처리 구축:** JIRA 이슈 변경 시 즉각적으로 데이터를 업데이트하기 위한 웹훅 수신 로직을 개발합니다.
- **데이터베이스(DB) 스키마 설계 및 구축:** 이슈 정보, 상태 이력, 사용자 정보를 효율적으로 저장할 테이블 구조를 설계합니다.
- **데이터 파싱 및 정규화 로직 구현:** JIRA에서 넘어온 원본 데이터를 대시보드 형식에 맞게 변환하고 가공하는 로직을 작성합니다.
- **대시보드 통계 지표 산출 로직 개발:** 진행률, 담당자별 부하량, 리드타임 등 시각화에 필요한 수치 계산 알고리즘을 구현합니다.
